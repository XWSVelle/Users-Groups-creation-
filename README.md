# Users-Groups-creation-

## Objective
Demonstrate role-based access control using IAM users and groups.

## User Created
- Administrator: full administrative access

## Groups Created
- DevOps: building, testing, and deploying softwares.
- ReadOnly: view-only access

## Design Choice
Groups were used to represent roles. Each group currently contains one
user for demonstration purposes. 


## Security Concepts Learned
- Role-based access control (RBAC)
- Least privilege
- Permission inheritance
